zml by josephwecker

Concise markup language + templating system using Erlang inspired by haml, SLiP, sexp2xml, and tenjin. Expanding to be somewhat of a view-centric but highly optimized production web framework.

Dependencies

* Erlang/OTP * Build script uses Python * Has not been tested on windows

Install

* Download source * In code/ directory execute "build" * This currently creates several beam files in the ebin directory * No system installation yet.
